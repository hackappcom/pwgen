pwgen
===

A simple implementation of pwgen in python

Installation
---------

    pip install pwgen
    easy_install pwgen

Basic Usage:
------------------

    from pwgen import pwgen
    pwgen()
    >> '[^%6JK<&Lb}{.T#eYY+
    pwgen(10, no_symbols=True)
    >> ujr20YVkH3
    pwgen(50, 10, no_symbols=True, no_capitalize=True)
